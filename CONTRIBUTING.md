# Contributing to Farcastermini

We're glad you're interested in contributing to the Farcastermini project! Here are the guidelines to help you get started.

## Setting Up Your Development Environment

1. **Clone the Repository**  
   Use Git to clone the repository to your local machine:
   ```bash
   git clone https://github.com/Urea46/farcastermini.git
   cd farcastermini
   ```

2. **Install Dependencies**  
   Make sure to install the necessary dependencies. For Node.js projects, you might use:
   ```bash
   npm install
   ```

3. **Run the Project**  
   To start the development server, run:
   ```bash
   npm start
   ```

## Making Changes

1. **Create a Branch**  
   Before making changes, create a new branch for your feature or fix:
   ```bash
   git checkout -b feature/your-feature-name
   ```

2. **Make Your Changes**  
   Implement your changes with clear and concise code.

3. **Test Your Changes**  
   Ensure everything works as expected by running tests or the application.

## Submitting Pull Requests

1. **Commit Your Changes**  
   After testing, commit your changes with a descriptive message:
   ```bash
   git add .
   git commit -m "Your descriptive commit message"
   ```

2. **Push to Your Branch**  
   Push your changes to the remote repository:
   ```bash
   git push origin feature/your-feature-name
   ```

3. **Open a Pull Request**  
   Go to the repository on GitHub and click on the 'Pull requests' tab. Click on 'New pull request' and select your branch. Fill out the pull request template if available, and submit!

## Code of Conduct

By participating in this project, you agree to abide by the principles of our code of conduct.

Thank you for considering contributing to Farcastermini! We're looking forward to your contributions!
