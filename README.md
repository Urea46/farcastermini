# Project Title
Farcaster Mini

## Project Description
Farcaster Mini is a lightweight project that serves as an initial implementation of a decentralized communication protocol. It aims to provide users with secure, private, and reliable messaging capabilities.

## Features
- **Decentralized Architecture:** Eliminates the need for centralized servers.
- **End-to-End Encryption:** Ensures that messages are securely transmitted and can only be read by the intended recipient.
- **User-Friendly Interface:** Designed for ease of use for both technical and non-technical users.

## Installation
To install Farcaster Mini, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Urea46/farcastermini.git
   ```
2. Navigate to the project directory:
   ```bash
   cd farcastermini
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
